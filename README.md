# 🥗 Diet Recommendation System (ML Project)

This project uses a Decision Tree Classifier to recommend a diet plan based on user inputs like age, BMI, activity level, and fitness goals.

## 🚀 Features
- Predicts diet plan (Low Carb / Balanced / High Protein)
- Calculates BMI automatically
- Suggests food items

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
